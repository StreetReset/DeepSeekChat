# DeepSeekChat 

## Описание

Консольный ИИ-ассистент с поддержкой веб-сёрфинга, разворачиваемый полностью локально через связку Ollama и DeepSeek-R1. Проект разработан с акцентом на безопасность: встроенные алгоритмы очистки веб-страниц парсят безопасные текстовые слепки из поисковой выдачи, обогащая контекст модели свежими данными без риска для приватности пользователя.

## Стек

- Python
- Ollama
- DeepSeek-R1 (1.5b / 7b / 14b)
- Requests
- PyInstaller
- uv

## Установка зависимостей

Проект использует современный менеджер пакетов `uv`. Для настройки виртуального окружения и автоматической установки всех необходимых библиотек (включая `requests`) выполните команду в корневом каталоге проекта:

```bash
uv sync
```

### Настройка Ollama и ИИ-модели

1. Убедитесь, что у вас установлен **Ollama** с официального сайта [ollama.com](https://ollama.com/), который необходим для локального запуска нейросети.
2. Запустите `Windows PowerShell` и выполните команду для загрузки базовой модели:

   ```bash
   ollama run deepseek-r1:14b
   ```

> [!TIP]
> **Для слабых ПК:** Если ваш компьютер не подходит под системные требования версии 14B, рекомендуется использовать менее требовательные модели. Замените команду выше на:
> * `ollama run deepseek-r1:7b` (оптимально для большинства ПК)
> * `ollama run deepseek-r1:1.5b` (для слабых ноутбуков и видеокарт)

| **1.5B** | 4–6 ГБ | 8 ГБ | NVIDIA RTX 3050 / 4050, Apple M-чипы |
| **7B / 8B** | 8–12 ГБ | 16 ГБ | NVIDIA RTX 3060 / 4060, Apple M1/M2/M3 |
| **14B** | 16–24 ГБ | 32 ГБ | NVIDIA RTX 3090 / 4080 / 4090 |
| **32B** | 24–32 ГБ | 64 ГБ | 1 × RTX 4090 (с квантованием) или 2 × RTX 3090 |
| **70B** | 40–48 ГБ | 64–128 ГБ | 2 × RTX 3090 / 4090, NVIDIA A6000, Mac Studio |
| **671B** *(Full)* | 320 ГБ – 1.3 ТБ | 512 ГБ+ | Кластер из 8 × NVIDIA A100 / H100 |


## Запуск

Запустить чат-бота можно тремя способами и через `uv`:

```bash
# Вариант 1: Сборка и запуск готового приложения
# Откройте проводник, запустите файл build_app.bat (он автоматически настроит зависимости) 
# и откройте появившийся файл DeepSeekChat.exe внутри папки dist/

# Вариант 2: Запуск напрямую через uv в изолированном окружении
uv run python main.py

# Вариант 3: Классический запуск (если виртуальное окружение уже активировано)
python main.py
```

## Команды

Бот поддерживает следующие команды в консоли:

1. `<ваш вопрос>` на прямую от пользователя.
2. `/web <ваш вопрос>` — ищет актуальную информацию в интернете и отвечает на её основе. ИИ не переходит на сайты напрямую, а обрабатывает очищенную HTML-выжимку — это гарантирует полную безопасность при веб-сёрфинге.
3. `/prompt` — показывает текущий системный промпт, используемый для генерации ответов.
4. `exit` / `quit` / `выход` — завершает работу чат-бота.


## Возможности приложения

- **Локальный ИИ:** Полностью автономная работа на базе моделей `DeepSeek-R1` через Ollama.
- **Умный веб-поиск:** Интеграция с DuckDuckGo для получения актуальной информации из сети в реальном времени.
- **Безопасный сёрфинг:** Автоматическая очистка и фильтрация HTML-кода со сторонних сайтов перед передачей данных в ИИ.
- **Гибкое управление промптами:** Динамическое формирование контекста в зависимости от режима работы (с доступом в интернет или без).

## Пример использования

Запустите скрипт и начните диалог:

```
Ты: Привет, как дела?
ИИ: Дела хорошо, как у тебя?

Ты: /web Погода в Москве
Ищу в интернете...
ИИ: В Москве сегодня погода солнечная. Температура +18°C.
```
