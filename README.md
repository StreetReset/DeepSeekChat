# DeepSeekChat

## Описание

Консольный ИИ-ассистент с поддержкой веб-поиска, работающий локально через связку Ollama и DeepSeek-R1. Проект умеет отправлять обычные вопросы в локальную модель, а по команде `/web` искать информацию через DuckDuckGo и добавлять найденные фрагменты в контекст ответа.

## Стек

- Python
- Ollama
- DeepSeek-R1
- DuckDuckGo HTML
- PyInstaller
- uv

## Установка зависимостей

Проект использует менеджер пакетов `uv`. Для настройки виртуального окружения и установки зависимостей выполните команду в корне проекта:

```bash
uv sync
```

### Настройка Ollama и модели

1. Установите **Ollama** с официального сайта: [ollama.com](https://ollama.com/).
2. Загрузите модель, которая используется в `main.py`:

   ```bash
   ollama pull deepseek-r1:14b
   ```

> [!TIP]
> Если 14B работает медленно, можно заменить модель в `main.py`:
> - `deepseek-r1:7b`
> - `deepseek-r1:1.5b`

### Системные требования для моделей DeepSeek-R1

| Версия модели | Мин. видеопамять (VRAM) | Мин. оперативная память (RAM) | Рекомендуемое железо / GPU |
| :--- | :--- | :--- | :--- |
| **1.5B** | 4-6 ГБ | 8 ГБ | NVIDIA RTX 3050 / 4050, Apple M-чипы |
| **7B / 8B** | 8-12 ГБ | 16 ГБ | NVIDIA RTX 3060 / 4060, Apple M1/M2/M3 |
| **14B** | 16-24 ГБ | 32 ГБ | NVIDIA RTX 3090 / 4080 / 4090 |
| **32B** | 24-32 ГБ | 64 ГБ | 1 x RTX 4090 с квантованием или 2 x RTX 3090 |
| **70B** | 40-48 ГБ | 64-128 ГБ | 2 x RTX 3090 / 4090, NVIDIA A6000, Mac Studio |
| **671B** *(Full)* | 320 ГБ - 1.3 ТБ | 512 ГБ+ | Кластер из 8 x NVIDIA A100 / H100 |

## Запуск

Запустить чат-бота можно через `uv`:

```bash
uv run python main.py
```

Или классическим способом, если виртуальное окружение уже активировано:

```bash
python main.py
```

## Сборка в exe

Для сборки приложения запустите:

```bash
build_app.bat
```

После сборки файл появится в папке:

```text
dist/DeepSeekChat.exe
```

## Команды

Бот поддерживает следующие команды в консоли:

1. `<ваш вопрос>` — обычный вопрос к локальной модели.
2. `/web <ваш вопрос>` — ищет информацию в интернете через DuckDuckGo и отвечает на основе найденных фрагментов.
3. `/prompt` — показывает текущий системный промпт.
4. `exit` / `quit` / `выход` — завершает работу чат-бота.

## Возможности приложения

- **Локальный ИИ:** работа на базе моделей `DeepSeek-R1` через Ollama.
- **Веб-поиск:** получение кратких фрагментов выдачи DuckDuckGo.
- **Очистка HTML:** удаление тегов и лишних пробелов перед передачей текста модели.
- **Гибкий промпт:** системные правила ответа хранятся в `main.py`.

## Пример использования

Запустите скрипт и начните диалог:

```text
Ты: Привет, как дела?
ИИ: Хорошо, спасибо. Чем помочь?

Ты: /web Погода в Москве
Ищу в интернете...
ИИ: По найденной информации...
```
